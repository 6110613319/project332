### About Project
* เป็นการจำลองร้านขายแฮมเบอร์เกอร์โดยมีเครื่องดื่ม อาหารว่าง และรายการเบอร์เกอร์ให้เลือก 2 แบบคือ
    - แบบจัดเป็นเซต
    - แบบการเลือกตามใจ
### Pattern Design
* Decorater Pattern
    - เบอร์เกอร์แบบเลือท๊อปปิ้งเพิ่ม 
* Strategy Pattern
    - เครื่องดื่ม และ อาหารว่าง
* Builder Pattern
    - จัดโปรโมชั่น
* Template Pattern
    - จ่ายเงิน
